# CustomGUI - Fabric 1.21.4 port

(Собран на базе версии под 1.21.1 — зависимости в `gradle.properties`
обновлены на актуальные под 1.21.4: yarn 1.21.4+build.8, fabric-api
0.113.0+1.21.4, loader 0.16.9. Сигнатура mixin-метода
`renderFirstPersonItem` между 1.21.1 и 1.21.4 не менялась по данным
changelog Fabric, но я не компилировал это здесь — см. раздел "Важно"
ниже, если Loom всё же на неё пожалуется.)


Порт мода Custom Hand Gui (оригинал под Forge 1.12.2, автор AqUpd) на
Fabric для Minecraft 1.21.1. Функциональность (сдвиг/поворот руки от
первого лица, ползунки конфига, хоткеи, сдвиг чата) сохранена, весь
код переписан под современные Fabric API и маппинги (Yarn 1.21.1).

## Важно

Этот код НЕ был скомпилирован и протестирован в игре (в песочнице, где
он написан, нет доступа к репозиториям Minecraft/Fabric). Возможны
мелкие ошибки компиляции, особенно в сигнатуре mixin-метода
`renderFirstPersonItem` — если Minecraft версии, которую ты используешь,
отличается от 1.21.1 хоть на минорную версию, маппинги могли измениться.

## Как собрать

1. Установи JDK 21.
2. Открой проект в IntelliJ IDEA (Community достаточно) с плагином Minecraft
   Development, ИЛИ просто выполни в терминале из папки проекта:
   ```
   ./gradlew build
   ```
   (для этого сначала выполни `gradle wrapper` один раз, если папки
   `gradle/wrapper` нет — либо просто открой проект в IDEA, она сама
   всё подтянет через Loom).
3. Готовый jar появится в `build/libs/customgui-2.0.0.jar`.
4. Если сборка упадёт на mixin-е (`renderFirstPersonItem` not found) —
   открой `HeldItemRenderer.class` через IDEA (Ctrl+клик по классу в
   декомпилированном виде) и сверь точное имя/сигнатуру метода,
   отвечающего за рендер предмета от первого лица, поправь строку
   `method = "..."` в `MixinHeldItemRenderer.java`.

## Что перенесено

- Сдвиг/поворот руки (`MixinHeldItemRenderer`, аналог `MixinItemRenderer` из 1.12.2)
- Хоткеи: смена главной руки, открыть конфиг, сброс, рандомизация
- Конфиг-экран с ползунками (переписан на vanilla `SliderWidget`,
  так как `GuiPageButtonList` из 1.12.2 в игре больше не существует)
- Сохранение настроек в файл (тот же формат, что и в оригинале)

## Чего нет (можно добавить)

- Сдвиг позиции чата при левше (`MixinGUIChat` из оригинала) — пропущен
  в этой версии, т.к. `ChatHud` в современных версиях устроен иначе и
  требует отдельной проверки метода для mixin. Если нужно — скажи,
  добавлю отдельным шагом.
